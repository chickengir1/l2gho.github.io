1. 엘리스 스토어

지난 3월부터 9월까지 KDT 주관 엘리스 트랙 부트캠프에 참여하며 프론트엔드 개발을 주도적으로 담당했습니다. UI/UX 설계부터 실제 기능 구현, 그리고 서버 배포 작업까지 다방면에서 기여하며 팀 프로젝트의 성공을 이끌었습니다.

직관적이고 사용성을 강조한 UI/UX 설계

  • 깔끔한 레이아웃: 유저 정보, 내비게이션 메뉴, 상품 목록을 명확히 구분하여 가독성 향상.
  • 직관적 인터페이스: 상품 삭제, 총 금액 확인, 쇼핑 진행 등 주요 기능을 쉽게 파악하도록 구성.
  • 반응형 디자인: 다양한 기기 환경에서 일관된 사용자 경험 제공.
  • 효율성 강화: 최소한의 클릭으로 작업 완료가 가능하도록 워크플로우 설계.

배송 관리 UI 설계 및 구현

  • 배송 상태 시각화: 주문 상태(배송 중, 배송 완료)를 명확히 구분하여 사용자가 빠르게 파악 가능.
  • 상품 정보 표시: 주문 날짜, 제품명, 사이즈, 색상 등 정보를 카드 형태로 구성.
  • 액션 버튼 제공: "주문 취소", "주문 수정" 버튼으로 빠른 요청 처리 가능.

상품 관리 대시보드 UI 설계 및 구현

  • 종합 정보 제공: 브랜드 수와 재고 수를 강조하여 한눈에 확인 가능.
  • 상품 목록 관리: 상품 이미지, 이름, 가격 등 정보를 표 형태로 제공.
  • 직관적인 수정/삭제 기능: 각 상품별 수정 및 삭제 버튼 제공.
  • 상품 추가 기능: 새 상품을 간편히 등록할 수 있는 기능 추가.

2. 모바일 UI 디자인

최근 진행한 개인 프로젝트에서 메인 페이지와 예약 관리 페이지의 모바일 UI 설계 및 구현을 주도적으로 담당했습니다. 사용자 중심의 설계로 주요 기능 접근성을 개선하고, 효율적인 UI를 제공했습니다.

예약 관리 모바일 UI 설계 및 구현

  • 날짜별 예약 정리: 사용자가 원하는 정보를 쉽게 찾을 수 있도록 예약 목록을 날짜별로 정리.
  • 상세 예약 정보 표시: 가게 이름, 위치, 예약 시간 등 정보를 카드 형태로 구성.
  • 유저 액션 버튼 제공: "리뷰 쓰기" 버튼으로 예약 후 피드백 작성 가능.

메인 페이지 모바일 UI 설계 및 구현

  • 추천 섹션 구성: 사용자 선호에 맞춘 추천 장소를 카드 형태로 제공.
  • 카테고리별 분류: 인기 장소, 이전 방문 장소 등으로 분류하여 탐색 편의성 강화.
  • 장소 상세 정보 표시: 가게 이름, 위치, 키워드 등을 명확히 제공하여 탐색 효율성 향상.
